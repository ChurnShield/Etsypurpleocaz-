## 🚨 Troubleshooting

### Problem: "npm is not recognized"

**Solution:**
Your computer doesn't have Node.js installed. Let's install it:

1. Go to: https://nodejs.org/
2. Click the big green button "Download for Windows"
3. Run the installer
4. Restart your terminal
5. Try `npm run dev` again

### Problem: "Cannot find module"

**Solution:**
You need to install dependencies first:

1. In terminal, type: `npm install`
2. Wait for it to finish (takes 1-2 minutes)
3. Then try `npm run dev` again

### Problem: Browser shows error page

**Solution:**
1. Check if terminal is still running `npm run dev`
2. Look for the Local URL in terminal (should be `http://localhost:5173/`)
3. Make sure you're using that exact URL

### Problem: Can't log in to dashboard

**Solution:**
1. Go to: `http://localhost:5173/auth`
2. Click "Sign Up"
3. Create an account with any email/password
4. Verify email if needed
5. Then go to dashboard
