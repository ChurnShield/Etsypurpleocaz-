## ЁЯУ╖ Screenshot Tips

**For Best Quality:**
1. Make your browser window as large as possible
2. Zoom to 100% (press `Ctrl + 0`)
3. Select the entire visible area (not just a small part)
4. Save as PNG (not JPG - PNG is clearer)

**What to Capture:**
- Capture the ENTIRE page/interface shown in the HTML file
- Don't crop too tightly - include some padding around edges
- Make sure all text is readable
