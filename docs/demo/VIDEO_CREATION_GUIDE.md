# 🎬 Creating a ChurnShield Cancel Flow Widget Video

## Quick Start Guide

I've created everything you need to showcase the cancel flow widget:

### 📁 Files Created:
- **`cancel-widget-video-script.md`** - Complete video script with timing and visuals
- **`cancel-widget-demo.html`** - Interactive demo page showing the full flow

### 🎯 What the Widget Does:
1. **Exit Survey** - Asks customers why they're canceling
2. **Smart Offers** - Shows personalized retention offers (discounts/pauses)
3. **Auto-Apply** - Integrates with Stripe to apply offers instantly
4. **Analytics** - Tracks saves and revenue impact

### 🎥 How to Create the Video:

#### Option 1: Screen Recording (Easiest)
1. Open `cancel-widget-demo.html` in your browser
2. Use Windows Game Bar (`Win + G`) or OBS Studio to record
3. Click through each step while narrating
4. Edit with simple transitions and text overlays

#### Option 2: Use Mockup Screenshots
1. Open each HTML file in `ChurnShield-Launch/screenshots/`
2. Take screenshots (Win + Shift + S)
3. Create a slideshow video with voiceover

#### Option 3: Live Demo (Advanced)
1. Set up the actual widget with test Stripe credentials
2. Record a real customer flow
3. Show the Stripe dashboard updating

### 🛠️ Tools Recommended:
- **Recording:** OBS Studio (free) or Windows Game Bar
- **Editing:** CapCut (free) or DaVinci Resolve (free)
- **Voiceover:** Built-in Windows Voice Recorder
- **Music:** Free background music from YouTube Audio Library

### 📋 Video Checklist:
- [ ] Clear introduction explaining the problem
- [ ] Show the exit survey screen
- [ ] Demonstrate personalized offers
- [ ] Highlight Stripe integration
- [ ] Show analytics dashboard
- [ ] End with clear call-to-action
- [ ] Keep under 3 minutes
- [ ] Professional voiceover and music

### 🚀 Pro Tips:
- Use fullscreen browser (F11) for clean recordings
- Add zoom effects on important elements
- Include text overlays for key statistics
- Test the audio levels before recording
- Save in MP4 format for maximum compatibility
